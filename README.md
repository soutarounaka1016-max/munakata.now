# むなかたNOW

宗像市のニュース、イベント、防災、天気、交通への入口を一画面にまとめる地域情報Webアプリです。

## 公開URL

https://soutarounaka1016-max.github.io/munakata.now/

## 主な機能

- ルールベースの重要度で選ぶ「今日知るべき3件」
- 宗像市の天気と3日予報
- 市役所・宗像ユリックス中心の新着情報
- タイトル・本文・情報源・カテゴリの横断検索
- 今日・今週・今月・すべてのイベント絞り込み
- 月表示イベントカレンダー
- OpenStreetMapによる施設・会場マップ
- 宗像市、防災、宗像ユリックス、JR九州、福岡県、図書館、ごみ、学校への公式リンク
- 記事とイベントの端末内保存（localStorage）
- 前回閲覧後の新着を表示する端末内通知センター
- PWA用マニフェストとオフラインキャッシュ
- 6時間ごとの情報更新とGitHub Pagesへの自動再公開

## 情報の扱い

このアプリは非公式です。タイトルと短い整理文を表示し、詳細は必ず公式サイトへリンクします。災害、交通、行政手続きなど重要な判断はリンク先の最新情報を確認してください。

## 検証

- データ構造・URL・日付・重複の検証
- Playwright ChromiumでiPad横向き・スマートフォン表示を確認
- 検索、カレンダー、地図、記事保存、イベント保存を自動操作
- 重大なJavaScriptエラーなし
- 公開URLのHTTP 200と画面タイトルをGitHub Actionsで確認
- 情報更新からGitHub Pages再公開まで実運転で確認

詳細は `STATUS.md` を参照してください。
