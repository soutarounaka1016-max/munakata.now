# むなかたNOW

宗像市のニュース、イベント、新店舗、限定メニュー、新商品、セールなどを一画面にまとめる地域情報Webアプリです。

地域店だけでなく、宗像市内に公式店舗が確認できるチェーン店の情報も掲載します。詳細や販売状況は必ず公式リンクで確認できる構成です。

## 公開URL

https://soutarounaka1016-max.github.io/munakata.now/

## 主な機能

- ルールベースの重要度で選ぶ「今日知るべき3件」
- Workers AIによる「今日の宗像おすすめ」
- 宗像市内のチェーン店舗一覧
- チェーン店の限定メニュー、イベント、新商品、セール・キャンペーン
- 情報ごとのAI要約とAIタグ
- 今日確認する価値を整理するAIランキング
- 選択した2〜3件のAI比較
- AIが利用できない場合のルールベース代替
- 宗像市の天気と3日予報
- 市役所・宗像ユリックス中心の新着情報
- タイトル・本文・情報源・カテゴリの横断検索
- イベントの期間絞り込みとカレンダー
- 施設・会場の地図導線
- 記事とイベントの端末内保存
- 前回閲覧後の新着を表示する端末内通知センター
- PWA用マニフェストとオフラインキャッシュ

## AI機能

Cloudflare Workers AIを利用し、次の4用途を1つのWorkerで提供します。

- `recommend`: 条件に合う情報を最大3件提案
- `enrich`: 掲載情報を短く要約し、タグと注意点を生成
- `rank`: 新しさ、終了時期、公式確認、限定性などを踏まえて順位付け
- `compare`: 2〜3件を利用場面、強み、確認点で比較

AIには掲載済みの候補だけを渡し、候補外のIDや入力にない情報を採用しないようJSON Schemaとサーバー側検証を行います。

## 情報の扱い

このアプリは非公式です。タイトルと短い整理文を表示し、詳細は公式サイトへリンクします。

チェーン店の全国企画は、宗像市内に公式店舗が確認できるブランドだけを掲載します。ただし、店舗ごとの販売、在庫、価格、実施状況は異なる場合があるため、公式ページや店頭で確認してください。

災害、交通、行政手続きなど重要な判断も、リンク先の最新情報を確認してください。

## 検証

- JSON構造、URL、日付、重複、店舗と企画の関連を検証
- Workers AIの4用途を単体テスト
- 公開Workerへ実リクエストを送り、4用途すべてを確認
- Playwright WebKitで公開URLを操作
- チェーンカード表示、AIランキング、AI比較、AI要約・タグを自動確認
- 重大なJavaScriptエラーがないことを確認

詳細は `STATUS.md` を参照してください。
